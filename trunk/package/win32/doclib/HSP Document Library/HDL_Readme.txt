■ 概要
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP Document Library (HDL) は、HSP のドキュメントをすべてまとめて 検索・閲覧 す
るためのブラウザです。

従来の HSP ドキュメントは、hs ヘルプ, txt/HTML ドキュメント, サンプルスクリプト
の 3 種類に分断されており、総合的なマニュアル検索やデータの相互参照ができません
でした。HDL では、これらのファイルすべてを網羅して検索することができます。

ハイライト
・プラグイン・モジュールごとにドキュメントをカテゴリ化できる仮想ドキュメントツリー
・リファレンス、ドキュメント、サンプルをまとめて検索する クロスサーチ
・hs ドキュメンテーションコメントつきの HSP ヘッダファイルのサポート



■ インストール方法
【▲注意】従来の HSP HELP Browser とはインストール方法が異なります。
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP Document Library を使用するためには、HSP インストールディレクトリに以下のフ
ァイルをコピーします。

【▲注意】下記のファイルを、hsp3.exe と同じディレクトリにコピーします。

・hdl.exe
・sqlite3.dll
・hdlmain.css
・hdlmenu.css
・doclib ディレクトリ

その後 hdl.exe を起動すると、ドキュメントを検索するためのデータベースが自動的に
作成され、HDL が使用可能になります。文字コードの問題で読み出せないファイルがある
場合は、データベース作成時にエラーダイアログで通知されます。

【▲注意】HDL をワンキーヘルプで使用するには下記の作業が必要です。

HDL をワンキーヘルプのサーバとして利用するためには、HDL のアーカイブに含まれる
hsphelp\helpman.exe で、HSP インストールディレクトリにある既存の hsphelp\
helpman.exe を置き換えてください。

ワンキーヘルプのプログラム名を切り替えられるエディタを使用している場合は、
ヘルププログラムとして hdl.exe を直接指定してください。

■ 動作環境について
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HDL は、Windows 95, Windows NT 4.0 およびそれ以前のバージョンの Windows では動作
しません。

初期バージョンの Windows 98 (+ IE4.0) で動作させるためには、OS コンポーネントの
更新が必要となります。

インストールするコンポーネント
・DCOM98 (http://www.microsoft.com/japan/com/dcom/dcom98/dcom13.aspx)
・atl.dll (http://activex.microsoft.com/controls/vc/atl.cab)



■ HSP 3.1 付属ドキュメントとの相性
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HSP 3.1 に付属のドキュメントには、そのままでは HDL では読み込めないものがありま
す。

・「初心者のための HSP 入門」は、文字エンコードが Shift_JIS でないため読み込めま
  せん。

・HSPLet のドキュメントは HSP インストールフォルダの docs 以外のディレクトリに置
  いてあるため、そのままでは HDL では表示されません。また、HSPLet ドキュメントの
  うち、  文字エンコードが Shift_JIS でないものは、適切なディレクトリに置いたと
  しても  そのままでは読み込むことができません。



■ アンインストール
￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣￣
HDL によってコンピュータに保存された設定ファイル、データファイルを消去する場合は、
同梱の hdl_uninst.bat を実行してください。

HDL をアンインストールする場合は、hdl_uninst.bat を実行後、インストール時にコ
ピーしたファイルをすべて削除してください。
